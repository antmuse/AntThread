enum irr::EMutexType

Mutex type defines.

枚举值
EMT_RECURSIVE 

Recursive mutex, that is, the same mutex can be locked multiple times by the same thread (but, of course, not by other threads).

EMT_NO_RECURSIVE 

Not recursive mutex, but more faster than recursive mutex.
